# 📧 Spam Email Classifier

A simple machine learning project that classifies SMS messages as **Spam** or **Ham** using a Naive Bayes classifier and TF-IDF vectorization.

## 🚀 Features

- Uses a labeled dataset of 5,500+ SMS messages
- Preprocesses text using NLTK and scikit-learn
- Trains a Naive Bayes classifier
- Outputs model accuracy and a detailed classification report

## 📁 Dataset

The dataset is based on the [UCI SMS Spam Collection Dataset](https://archive.ics.uci.edu/ml/datasets/SMS+Spam+Collection), loaded via a TSV file.
